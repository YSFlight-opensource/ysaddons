
　〜AceCombat〜

AceCombat5、zeroに登場した機体達です。


・F-14A
AC5のラーズグリーズ隊仕様。
016が主人公であるブレイズ機、007がケイ・ナガセ（最終階級大尉）の機体となっています。
3番機、4番機の再現用に番号がついていない機体も製作しましたのでご活用ください。

・JAS-39C
ACzeroのベルカ空軍エース部隊、インディゴ隊仕様。

・F-15C
ACzero、主人公である後に「円卓の鬼人」と呼ばれるTACネーム「サイファー」の搭乗機。

・F-15C
ACzero、主人公の相棒である「片翼の妖精」ことラリー・フォルク少尉の搭乗機。

・E-767
ACzeroで主人公機をサポートするAWACS「イーグルアイ」仕様。


この機体についてMasa-AT氏、及びバンダイナムコゲームズ株式会社に質問することは絶対におやめください。
