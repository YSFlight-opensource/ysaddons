*　概要　*
F-15バリエーションの偵察機バージョンとヴァルキリーシリーズの第二弾を製作してみました。
RF-15Jは単座コンフォーマルタンク装備の改C型的な位置付けになると思います。

VLKには光学迷彩を試験的に搭載させてみました、爆弾槽キーを押しますと起動します。
外部武装と搭載物に関しては、迷彩の効果範囲外ですので起動中は視認できますので注意してください。

*　機体　*
JP_RF-15J_ReconEagle01 (仁)
JP_RF-15J_ReconEagle02（義）
VLK_F-15C/SMT_Gondur

*　特別収録　*
ER_Su-37_SuperFlanker

エルジア空軍第156戦術戦闘航空団所属機モデルを収録しました。

色はGACに近づけてありますゆえ多少見劣りする部分があるかもしれませんがご了承ください。
また当機にはUmaibow氏作成のSu-37Yellow13から国籍マークを使わせていただいております。

当機体を使用して発生した損害及び不具合に関してYS Flight Simulation System 2000 製作者である
山川機長、またGeneralAirCombatAddon製作者、Masa-AT氏に苦情、質問および改善要求等を送らない
ようにお願いします。
