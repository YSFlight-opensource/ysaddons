*　概要　*
米国空・海軍及び日本国航空自衛隊版のF-16を製作しました。
今回選出されたのは特殊なものを含め3機種です。

*　機体　*
US_F-16Kai
US_F-16N
JP_F-16C

Kai型に関してはRCS軽減を実現したのみで、ウェポンベイ等は形だけになっております。
また、アップローダーの資料を基に作ってみただけですので形が異なる可能性があります。

当機体を使用して発生した損害及び不具合に関してYS Flight Simulation System 2000 製作者である
山川機長、またGeneralAirCombatAddon製作者、Masa-AT氏に苦情、質問および改善要求等を送らない
ようにお願いします。
