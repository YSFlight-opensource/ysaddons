*　概要　*
Valkyrieシリーズ第一弾　"Alvitr(アルヴィト)"

垂直尾翼に女の子がデザインされておりますが、その所為で多少重くなっております。
Valkyrieシリーズって名前はいいですが単に痛機です。
そのテのモノが苦手な方、或いは重たいのでちょっと･･･と言う方はGACv08、Su-30MKI.dnmの名前を
Su-30MK_ALVITR.dnmと変えるだけでOKです

*　機体　*
vlk_Su-30MK_Alvitr

CTRL+Fサーチする時に「vlk」と押すと簡単に出ます。


当機体を使用して発生した損害及び不具合に関してYS Flight Simulation System 2000 製作者である
山川機長、またGeneralAirCombatAddon製作者、Masa-AT氏に苦情、質問および改善要求等を送らない
ようにお願いします。
