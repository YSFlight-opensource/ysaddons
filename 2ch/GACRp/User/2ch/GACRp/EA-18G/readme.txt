GAC収録『US_F/A-18F_SuperHornet』改造機

BaseModel:US_F/A-18F_SuperHornet

EA-18G Growler

【改造項目】
1.翼端AAMランチャー（Station1,11）をAN/ALQ-218受信システムに変更
2.胴体下面（Station6）にAN/ALQ-99ジャミングポッド（低帯域型）を装備
3.主翼下（Station3,9）にAN/ALQ-99ジャミングポッドを装備
4.実機写真より、主翼前縁折り畳み機構付近の形状変更
5.AN/ALQ-218装備に伴い、主翼端の編隊灯形状変更
6.機体仕様、武装状況にあわせDAT変更
7.機体カラーを単色に変更
8.胴体側面、水平尾翼付近に『NAVY』表記、インテーク脇マークの追加




【武装】
AIM9：最大２（Station2,10）
AIM120：最大６（Station2,4,5,7,8,10）
AGM65：最大４（Station2,4,8,10）
B250：最大２（Station4,8）
B500：最大４（Station2,4,8,10）
GUN：なし

実機の武装は、ALQ-99と218で潰れたステーション以外Ｆ型と同じのようですが、
20mmは積んでないようです。
ECM機器を詰めちゃったそうなので、DATでもGUN:0になってますから注意です。

余談ですが、ALQ-218（翼端のやつね）の上に生えてる三角。
あれ空力特性だかを考慮して、AIM-9取付け時を参考に上下計８枚つけたらしいん
ですが、結局「あんましイラネ」って下面のは無くしたとか。
だから上にだけ斜めに４枚生えてるんだそうですよ。
