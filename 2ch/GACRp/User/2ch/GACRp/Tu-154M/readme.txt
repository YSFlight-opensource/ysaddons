・Tu-154M"FACT"(Repainted GAC_CIV_Tu-154M)
FACTはFuture Aircraft Control Testbedの略でそれが示すように
操縦システムのテストで使われたようです。外見は普通のTu-154Mと
変わりありませんが、コクピットは左側の席のみグラスコクピット化、
HUDの装備、操縦桿もスティック状になり真ん中と右側に二本あるなど
の改造が行われています。
