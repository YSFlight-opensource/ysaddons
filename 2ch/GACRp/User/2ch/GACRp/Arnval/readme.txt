
　〜MiG-35_Arnval〜

また武装神姫とのコラボです。
この機体についてMasa-AT氏、コナミ株式会社、島田フミカネ氏に質問することは絶対におやめください。

今回は天使型神姫アーンヴァルです。
アーンヴァルには限定品で黒いリペイント版が存在するので、それを再現する意味で黒いタイプも製作。
アーンヴァルはイタリア語で「突撃」なのでイタリア軍機にしようとしたのですが、どうもグッとくるのがなくて・・・
単機で「突撃」して戦えるMiG-35を選択。
機体側面の「FrontLine」は設定上のアーンヴァルのメーカー。
「01a」は実際のアーンヴァルの翼に書かれているものです。
