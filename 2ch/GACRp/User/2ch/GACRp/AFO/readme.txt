*　概要　*
特別要人輸送用機体集です。
アメリカ合衆国要人輸送機、VC-25等実在する要人輸送機をまとめてリペイントしてみました。
またこのパッケージ内にF-15改イーグルプラスを同梱しておりますが、一部外見を省いております。

*　機体　*
US_Boeing_747-2G4B_82-8000_AirForceOne　(合衆国大統領及び要人専用機)
US_E-4B_NEACP(国家空中作戦センター: National Airborne Operations Center)
JP_Boeing_747-400_JapaneseAirForceOne (日本国政府専用機)
JP_F-15Kai_Eagle+

要人輸送機3機に関して少々重くなってしまったので、重さが気になる、または不快に感じられる場合は
dnmファイルの差し替えを各自行ってください。こちら側では製作いたしません。
また、アップローダーの資料を基に作ってみただけですので形が異なる可能性があります。

当機体を使用して発生した損害及び不具合に関してYS Flight Simulation System 2000 製作者である
山川機長、またGeneralAirCombatAddon製作者、Masa-AT氏に苦情、質問および改善要求等を送らない
ようにお願いします。
