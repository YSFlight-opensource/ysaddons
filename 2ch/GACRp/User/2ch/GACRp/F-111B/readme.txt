・F-111B_Ardvark(Remodeled GAC_AU_F-111C_Ardvark)
F-111の海軍型。空母での運用にあわせ機首の切り詰め、主翼の面積拡大など改造した
結果空軍型との共通部分は３割程度となった上に重量が要求より１０トン以上オーバー。
機体も巨大となり、空母での運用は困難とされ海軍の強い反対もあり開発は中止となった。
このとき開発されたP&W TF-30エンジンやAIM-54、AGW-9レーダーはF-14に流用され。
F-111Bもその開発のためのデータ収集に使用され、このとき空母着艦試験も行われたが
特に問題は無かったとされている。
