F-4G USAF
・機体解説
F-4Gはベトナム戦争の教訓をうけてF-4Eをベースに開発されたSEAD（敵防空網制圧）専用機です。
F-4Eの機首に搭載されていたM61A1「バルカン」20mm機関砲を取り外し、AN/APR-47 「ELS」レーダー捜索用アンテナを設置。
さらに垂直尾翼先端にも同タイプのアンテナを設置し、レーダー捜索に特化させました。
主武装はAGM-88「HARM」対レーダーミサイルで、AN/APR-47 「ELS」で探知したレーダーをこれで攻撃、撃破します。
引退間近の1991年、湾岸戦争に参加し、イラク軍のレーダーサイトを多数破壊、アメリカ空軍のF-4系統機の有終の美を飾りました。
（ちなみに、損害は1機だけで、それも燃料切れのため脱出・放棄されたものであり、死者は０でした。）
