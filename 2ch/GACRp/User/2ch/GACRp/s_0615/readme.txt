今回は武装神姫コラボではなく、アニメ「スカイガールズ」に登場する一条瑛花さんを描いてみました。
一条さんの搭乗機であるG2N3雷神の武装を装備しています。
大型ビーム砲とガトリングを別々に射撃できるようにするために、
砲には通常の機銃を、ガトリングには自動機銃を割り当てました。
自動機銃は上4ﾟ、下3ﾟ、左右3ﾟの範囲に敵機を捕らえたときに作動します。
爆弾倉扉開閉操作でガトリングが回転するので、射撃とタイミングを合わせて回転させると格好良いかもです。


この機体についてMasa-AT氏、コナミ株式会社に質問することは絶対におやめください。
