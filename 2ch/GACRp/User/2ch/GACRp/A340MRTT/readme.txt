*　概要　*
架空ですがありえそうでありえない物を作ってしまいました。
コンセプトの元の機体はKC-10で当機体(A340-600MRTT)は実在しておりません。

*　機体　*
NATO_A340-600_MultiRoleTankerTransport

当機体に使用されているフライングブーム、またドローグホース、バディポッド、またそれに付属する
パイロンはSylph氏製作の

「JP_KC-767」
「US_MC-130P_Combat_Shadow」

から使用させていただいております。

またNATO+OTANマーキングにつきましてはTF58氏のGACリペイントパッケージ収録の

「E-3A_AWACS_NATO_{2ch/TF58}」

から使用しています。

* 給油方法 *
フライングブームは可動機銃設定ですので可動範囲内でなら自由に操作が可能です。またブーム先端部の
延長収縮に関しては爆弾槽キーで延長、収縮が可能です。
ドローグホースは355kt以上で展開します、355kt以下になりますと自動的に収納されますのでご注意ください。
また給油支援灯火は可変ノズル割り当てです、夜間飛行の際に受油側パイロットへの識別にご使用ください。

ようにお願いします。

これらを使用して得た不利益、損害等は一切責任を負わないものとします。
全て自己責任でご使用ください。

また加工されたスクリーンショットを使用した宣伝行為、再加工、転載を禁止するものとします。
