
　〜ITAV-8B〜

この機体についてMasa-AT氏、コナミ株式会社、及びBLADE氏に質問することは絶対におやめください。


当然と言えば当然ですが、デカールが重いです。
なのでデカールを撤去したcoarseファイルを用意しておきました。
それと、デカールがよく見える追加視点を2つ追加。
結構頑張ったので、暇があれば見てやってください。



読み方は「いたぶいえいとびー」です。
KO_ITAV-8Bの「KO」はKONAMIのKO。
主翼上面と機首に書かれている「Kemotech」はハウリンのメーカーです。
その横にあるシルエットはハウリンです、当然。
機体選択は、ハウリンの犬型と"harrier"という単語が持つもう一つの意味、「ハリアー犬」にかけたつもり。
あと、ハウリンは全射程「強襲」型神姫なので、「強襲」揚陸艦で運用されるハリアーを選んだってのもあります。
設定上は神姫は人間のパートナー・・・というわけで、パートナーが必要な複座機。
色は以前おふざけで白いTAVを作ったら結構いい感じだったのでそれを採用。
機体番号は・・・語るに及ばずですねｗ

とまぁ、TAV-8Bを選ぶにあたって色々なネタを含ませたわけですが、こうやって解説しないと誰も分からないというｗ
