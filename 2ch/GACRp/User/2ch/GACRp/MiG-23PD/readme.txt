・MiG-23PD_Faithless(Remodeled GAC_KP_MiG-23MLA_Flogger)
実験機23-31(MiG-21PD)のデータを基に作られたSTOL試作戦闘機で胴体内に
2基のリフトエンジンを装備している。
1967年に初飛行後、数回テストされるがリフトエンジンが飛行中は完全に
死重となることに加え、リフトエンジンを機体内部に入れるため燃料や機
材の搭載量が限定されることが判明。同時期に開発されていた可変翼試作
戦闘機23-11と比べてもSTOL性能以外では劣っており、将来の発展性も無い
と判断されたため、23-11がMiG-23として採用され当機は開発中止となった。
