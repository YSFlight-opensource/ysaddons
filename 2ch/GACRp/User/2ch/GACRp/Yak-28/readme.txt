・Yak-28L_Brewer(Remodeled GAC_KP_Il-28_Beagle)
軽爆撃型の一つで、IL-28Iの改良型。Yak-28Iは胴体下面に索敵用のレーダーを装備していたが、
そのレドームが邪魔で爆弾搭載作業に時間がかかりすぎるという欠点を持っていたため、それを
改善するためにレドームを取り外し、代わりにDBS2SLotosレンジ・ファインダー爆撃システムを
装備している。
レーダーを下ろしたため自立索敵能力は無くなってしまったが、その反面機体がクリーンになった
こととエンジンを防空戦闘機型と同じR-11AF2-300に換装したため、速度などの性能は向上している。


この機体に関してMasa-AT氏に質問をすることは絶対にお止めください。
