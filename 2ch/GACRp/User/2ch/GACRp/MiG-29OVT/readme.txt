*　概要　*
ロシア空軍の試験機、またはデモンストレーション機を模して製作しました。


*　機体　*
Mig-29OVT、Su-30LL、Mig-31E

OVTカラーはそのままですが、Su-30LLはテストパイロッツ仕様になっております。
またOVT及びLLに関しては全機失速機動が可能なレベルに調整してあります。

当機体を使用して発生した損害及び不具合に関してYS Flight Simulation System 2000 製作者である
山川機長、またGeneralAirCombatAddon製作者、Masa-AT氏に苦情、質問および改善要求等を送らない
ようにお願いします。
