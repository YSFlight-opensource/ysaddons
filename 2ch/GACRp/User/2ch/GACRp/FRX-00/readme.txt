-------------------------------------------------------------------------------------------

*　概要　*

戦闘妖精雪風に登場するFFR-31Sylphide/FRX-00Mave"Yukikaze"を製作してみました。


*　機体　*

FAF_FFR-31_Sylphide_baseline1
FAF_FRX-00_Mave_yukikaze


*　注意　*

-Sylphide
・航法灯はノズルダウンキーで点灯します。

-Mave_yukikaze
・航法灯はノズルダウンキーで点灯します。
・爆弾槽キーで主翼、尾翼、エンテ翼が格納状態に移ります。
・速度増加に伴ってノズルが絞られます。


またランディングギア展開時に自動的に着陸灯が点灯するように細工をいたしました。
