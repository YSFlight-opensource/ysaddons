Windows XP Service Pack 2でYSFLIGHTのネット対戦を利用するときの注意

www.ysflight.com
 

Windows XP Service Pack 2でYSFLIGHTのネット対戦を利用しようとすると、次のようなダイアログが出る可能性があります。

ここで、必ず「Unblock」（日本語版だと、多分「ブロックしない」とかそういう翻訳になっていると思う）を選んでください。

　

もしも、誤って「Keep Blocking」を選んでしまった場合、次の手順でブロックを解除してください。

  1. コントロールパネルのセキュリティセンターを開く。
  2. Windows Firewallをクリックする。
  3. 「Exceptions」タブ（多分日本語版では「例外」とか翻訳してあるんじゃないかと思う）を選んで、fsmain.exeとfsmaino.exeの横のチェックボックスをチェックしてください。ここのチェックがはずれていると、YSFLIGHTはネット対戦できません。
